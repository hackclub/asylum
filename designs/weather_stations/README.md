# weather station
Here's my submission for my weather station! Under the asylum "dragons_weather"

## Features:
- Displays:
    - Date
    - Temperature 
    - Settings
- 4 WS2812Bs for dynamic LED effects
- A switch for turning on and off
- Slider for settings
